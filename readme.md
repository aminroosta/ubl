### ubl

utility tools for ubl.

**Installation:**
```
npm install -g ubl
```


**Usage:**

```
Usage: ubl [options]

Options:
  -r, --reset                    reset & delete the user account info
  -b, --branch <jira_ticket_id>  creates a feature branch from a jira ticket ticket id
                                        example: ubl -b 31019
                                        example: ubl -b UB-31019
  -h, --help                     display help for command
```
